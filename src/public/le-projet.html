Présentation du projet

Objectif

Créer une IA capable de résoudre un labyrinthe en un minimum de coups, avec plusieurs niveaux de difficulté.

Équipe

  • Aymerick RAKOTOARIVONY : Lead Front
  • Yohann FREMONT : Coordinateur Front
  • Lilian MARIE-JOSEPH : DevOps Docker
  • Davidson CHARLOT : Dev IA
  • Nathan BAPIN : Lead Dev Back
  • Quentin AYRAL : Coordinateur Back
  • Idel SADI : Dev Full stack

Ambitions

  • Créer une IA performante.
  • Apprendre à travailler en groupe et synchroniser deux équipes.
  • Accroître les connaissances en IA, TypeScript, Docker et gestion d’API.

Installation & utilisation

  1. Cloner le projet : git clone https://github.com/Projet-Naturun-the-GOAT/Naturun_The_Goat_FRONT.git
  2. Installer les dépendances : npm install
  3. Lancer le projet : npm run dev
  4. Ouvrir dans le navigateur : http://localhost:3000/
